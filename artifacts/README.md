# AGICE Evidence Pack (v1)

**Patent Pending (USPTO Provisional)** — Application No. **63/978,753** — Filed **2026-02-09**.

This record contains:
- Evidence bundles (trajectory logs, manifests, replay metadata)
- Integrity anchors (hash‑chained event logs)
- Summary evidence reports
- A tiny validator script for integrity checks (pack‑included)

## Recommended use
1) Download the Zenodo archive
2) Unpack locally
3) Follow the validation instructions in the pack README (includes a `validate.py` script)
